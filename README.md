EXPENSE TRACKER:
A simple , user-frinedly **python** application designed to help users track their daily expenses and manage their budget. 
FEATURES :
**add expenses** : Record new spending with descriptions and amounts.
**view history** : See a list of all previosouly recorded expenses.
**data storage** : Saves your data to a file so it's there the next time you open the program.
HOW TO RUN :
1. Make sure you have python installed on your computer.
2. Download teh expense_tracher.py file from the repository.
3. open your terminal or command prompt.
4. Run the script
